
# MERN Employee Management System (Full Stack)

## Tech Stack
- MongoDB
- Express.js
- React (Vite)
- Node.js

## Project Structure
MERN_Employee_Project
- backend
- frontend

## How to Run

### Backend
cd backend
npm install
npm run dev

Backend URL: http://localhost:5000

### Frontend
cd frontend
npm install
npm run dev

Frontend URL: http://localhost:5173
